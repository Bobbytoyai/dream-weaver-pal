# Code of Conduct

Bobby est un produit pour enfants. On code et on collabore avec le même soin qu'on attend dans nos interactions avec les enfants.

## Notre engagement

Toute personne qui contribue, code, review, participe à une réunion ou pose une issue est traitée avec :

- **Respect** — pas d'insulte, pas de condescendance, pas de "tu devrais savoir ça".
- **Patience** — la plupart des questions "bêtes" révèlent une doc manquante.
- **Honnêteté** — un feedback direct vaut mieux qu'un feedback enrobé. Direct ne veut pas dire brutal.
- **Curiosité** — on cherche à comprendre avant de juger.

## Inacceptable

- Harcèlement de toute nature (genre, orientation, origine, religion, handicap, etc.).
- Commentaires démeants ou dégradants, publics ou privés.
- Trolling, attaques personnelles, langage ou imagerie sexualisée.
- Publication d'informations privées sans permission.
- Conduite qui pourrait raisonnablement être considérée comme inappropriée dans un contexte professionnel.

## Responsabilités

Les mainteneurs s'engagent à :
- Clarifier les standards quand des comportements problématiques émergent.
- Retirer, éditer, refuser des contributions qui ne respectent pas ce code.
- Bannir temporairement ou définitivement tout contributeur qui adopte un comportement inapproprié, menaçant, ou nuisible.

## Application

Signalements : `conduct@bobby.toys`. Confidentiel, traité sous 48 h.

Inspiré de [Contributor Covenant 2.1](https://www.contributor-covenant.org/), adapté à un contexte produit-enfant.
